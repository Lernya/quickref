React cheatsheet

Um ein neues React-Projekt einzurichten, folgen Sie diesen Schritten:

Schritt 1: Erstellen Sie ein neues React-Projekt mit Vite

a.Öffnen Sie Ihr Terminal: Öffnen Sie Ihr Terminal oder die Eingabeaufforderung.

b. Führen Sie den Befehl Vite Create aus: Verwenden Sie den folgenden Befehl, um ein neues Vite-Projekt zu erstellen. Sie können my-react-app durch Ihren Projektnamen ersetzen.

>npm create vite@latest

c. Navigieren Sie zum Projektverzeichnis:

>cd my-react-app

Schritt 2: installieren

>npm install

Schritt 3: Tailwind CSS installieren

>npm install -D tailwindcss postcss autoprefixer
>npx tailwindcss init -p

Schritt 4: Konfigurieren Sie Tailwind-CSS

a. Konfigurieren Sie tailwind.config.js: Öffnen Sie die Datei tailwind.config.js und aktualisieren Sie das Inhaltsfeld, um Ihre Vorlagendateien aufzunehmen. Es sollte wie folgt aussehen:

/** @type {import('tailwindcss').Config} */
module.exports = {
content: [ "./index.html", "./src/**/*.{js,jsx,ts,tsx}", ],
theme: { extend: {}, },
plugins: [], }

b. Fügen Sie Tailwind-Direktiven zu Ihrem CSS hinzu: Öffnen Sie die Datei src/index.css (oder erstellen Sie sie, falls sie nicht existiert) und fügen Sie die folgenden Zeilen hinzu:

@tailwind base;
@tailwind components;
@tailwind utilities;

Schritt 5: Installieren Sie DaisyUI

DaisyUI ist eine Tailwind CSS-Komponentenbibliothek. Sie können sie installieren, indem Sie ausführen:

>npm install daisyui

Schritt 6: Konfigurieren Sie DaisyUI in Tailwind

Um DaisyUI zu verwenden, müssen Sie es in die Datei tailwind.config.js aufnehmen. Aktualisieren Sie den Abschnitt plugins:

import daisyui from "daisyui"
module.exports = { //...
plugins: [ daisyui, ], }

Schritt 7: Starten Sie Ihren Entwicklungsserver

Nachdem nun alles eingerichtet ist, können Sie Ihren Entwicklungsserver starten:

>npm run dev
